Plataforma web de apoyo a cuidadores (TFG)

Rol: Back-end Developer

Desarrollo del back-end de una plataforma web orientada al apoyo de cuidadores y familias, centrada en la gestión integral de pacientes y sus responsables. El sistema permite la administración de usuarios, cuidadores, hijos/pacientes y medicamentos, con una base de datos relacional diseñada específicamente para reflejar la relación entre padres e hijos. Se implementó un calendario interactivo de medicación, donde el cuidador registra los medicamentos día a día y el menor puede visualizarlos de forma clara.
El proyecto incluye la lógica de negocio en PHP, el diseño y gestión de bases de datos en MySQL y la comunicación con un front-end desarrollado en Angular

Calculadora EVAU

Rol: Front-end Developer

Aplicación web para el cálculo de la nota de la EVAU, permitiendo al usuario seleccionar comunidad autónoma, convocatoria, asignaturas y ponderaciones. La aplicación calcula dinámicamente las notas sobre 10 y sobre 14, obteniendo las ponderaciones actualizadas según la comunidad seleccionada.
Incluye también la generación de un modelo de examen en función del año, comunidad autónoma y asignatura, ofreciendo una herramienta práctica y sencilla para estudiantes.
